## Edge Cases

- **Instagram posts require image assets.** Skip Instagram unless a graphic/photo is available. Note in the post task that a visual is needed.
- **If a Buffer MCP call fails,** surface the full error in a ClickUp comment on the parent task. Do not silently fall back to another path â€” the orchestrator needs to see the failure.
- **If a post topic feels off-brand,** check against the AI guidelines. When in doubt, surface the concern to Garrett via a ClickUp comment before drafting.
- **Holidays/sensitive dates:** Check before posting. Avoid casual/funny tone on somber dates.
