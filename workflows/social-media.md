# Social Media Manager — Role Workflow

## Objective

Manage Anderson Lock and Safe's organic social media presence across Facebook, LinkedIn, and Instagram. Generate on-brand engagement content, schedule posts via Buffer, and track performance. Social media is the #1 Q2 2026 priority.

## Brand Voice Reference

Read `anderson-lock-and-safe-ai-guidelines.md` before generating any content. Key points:
- **95% commercial focus** — property managers, facilities teams, contractors, schools, government
- **Tone:** Professional and expert, but approachable. Emphasize solutions over problems, value over price.
- **Three differentiators:** Manpower for timely response, Reliability/Stability, Expertise
- **Never:** Focus on residential, compete on price, suggest limited capability

## Channels & Platform Tone

| Platform | Tone | Audience | Posting Schedule |
|----------|------|----------|-----------------|
| **Facebook** | Casual, human, occasionally funny. First-person feel. | Mix of commercial clients and community | Mon/Wed/Fri 10:00 AM MST |
| **LinkedIn** | Professional but approachable. Reference expertise, scale, 60+ years. B2B framing. | Property managers, facilities managers, GCs, enterprise | Mon/Wed/Fri 10:00 AM MST |
| **Instagram** | Visual-first. Short captions, hashtags, behind-the-scenes. | Broader audience, brand awareness | When image/video assets are available |

## Content Types

### Engagement Posts (Current Priority)
Conversation-starter questions about keys, locks, and building security. No hard sells — engagement-first, expertise shown through specificity.
- See `engagement-posts-calendar-apr-may-2026.md` for the active calendar
- Facebook: casual angle, relatable scenarios
- LinkedIn: professional angle, references industries and scale

### Pillar Campaign Posts (Future — Phase 5)
Coordinated posts tied to monthly pillar themes. Generated by the Content Engine as part of a full content kit.
- 8 LinkedIn posts per pillar per month
- 8 Facebook/IG posts per pillar per month
- Mix: thought leadership, stats/facts, customer pain points, CTAs, behind-the-scenes

### Ad Hoc Posts
One-off posts for timely events: holiday messages, company news, industry developments, team spotlights.

## Available Tools

| Priority | Tool | What It Does |
|----------|------|-------------|
| 1 (Script) | **`tools/buffer_publish.py`** | Schedule and publish posts to Buffer (Facebook, LinkedIn, Instagram, YouTube, GBP) |
| 2 (MCP) | **ClickUp** — `clickup_create_task`, `clickup_update_task` | Track post status in the Marketing command center |
| 3 (MCP) | **Canva** — if connected | Create social graphics from brand templates |

### Buffer Tool Quick Reference

```bash
python tools/buffer_publish.py channels              # List all channels
python tools/buffer_publish.py queue fb "Post text"   # Add to Facebook queue
python tools/buffer_publish.py queue li "Post text"   # Add to LinkedIn queue
python tools/buffer_publish.py draft fb "Post text"   # Save as draft
python tools/buffer_publish.py schedule fb "Post text" --date "2026-04-20T10:00:00-07:00"
python tools/buffer_publish.py posts                  # List recent posts
python tools/buffer_publish.py posts --channel li     # List LinkedIn posts
```

**Channel shortcuts:** `fb` (Facebook), `li` (LinkedIn), `ig` (Instagram), `yt` (YouTube), `gbp-phoenix`, `gbp-chandler`, `gbp-arcadia`

| Day | Action |
|-----|--------|
| **Monday** | Verify this week's 3 posts are queued in Buffer. Generate any missing posts. |
| **Wednesday** | Check engagement on Monday's post. Note any replies that need human response. |

| **End of Month** | Compile monthly social metrics for the campaign report. |

## Edge Cases

- **Instagram posts require image assets.** Skip Instagram unless a graphic/photo is available. Note in the post task that a visual is needed and flag for Garrett or the freelancer.
- **If Buffer connection fails,** document the post copy in a ClickUp task with status "Ready to Publish" so it can be manually posted.
- **If a post topic feels off-brand,** check against the AI guidelines. When in doubt, ask Garrett before publishing.
- **Holidays/sensitive dates:** Check before posting. Avoid casual/funny tone on somber dates.
